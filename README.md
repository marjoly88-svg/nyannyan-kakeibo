# 家計簿アプリ - iOS公開手順

## 📱 アプリ情報
- **アプリ名**: 家計簿
- **Bundle ID**: com.cupids.kakeibo
- **バージョン**: 1.0.0

---

## ✅ しょーちゃんがやること（この順番で！）

### STEP 1: Apple Developer登録
1. https://developer.apple.com/programs/ にアクセス
2. 「Enroll」をクリック
3. Apple IDでサインイン
4. $99/年を支払い（約15,000円）
5. 承認まで最大48時間待つ

### STEP 2: Codemagicアカウント作成
1. https://codemagic.io/ にアクセス
2. 「Start building」をクリック
3. GitHubアカウントでサインアップ（なければGitHubも作る）

### STEP 3: GitHubにプロジェクトをアップ
1. GitHubで新しいリポジトリ作成（名前: kakeibo-app）
2. このフォルダをアップロード

### STEP 4: CodemagicとApple Developerを連携
1. Codemagicにログイン
2. 「App Store Connect API」を設定
3. Apple Developerで証明書を作成
4. Codemagicに証明書をアップロード

### STEP 5: ビルド実行！
1. Codemagicでリポジトリを選択
2. 「Start new build」をクリック
3. 待つ（10〜20分）
4. 成功したらApp Store Connectに自動アップロード！

### STEP 6: App Store申請
1. App Store Connectにログイン
2. アプリ情報を入力
   - スクリーンショット（5.5インチ、6.5インチ）
   - 説明文
   - キーワード
   - プライバシーポリシーURL
3. 「審査に提出」をクリック
4. 審査待ち（1〜7日）
